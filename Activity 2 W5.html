Colors of the Rainbow

R

Red is a vivid color with the hex code #FF0000, one of the major colors in the RGB color model. A color associated with passion and love, danger and courage, red is an important staple in art, fashion, and the natural world.



O

Orange is a bright and vivacious color with the hex code #FF7E00. People often associated orange with optimism, confidence, enthusiasm, warmth, and agreeableness.



Y

Yellow is a cautionary color with the hex code #FFFF00. It symbolizes optimism, energy, joy, happiness and friendship. It might also stand for intellect. On the contrary, yellow can indicate jealousy, betrayal, illness and danger.



G

Green is the color of life with the hex code #006400. It evokes a feeling of hopefulness, responsibility, wealth, forgiveness, comfort and energy



B

Blue is often associated with sadness in the English language with the hex code #00008B. Blue is also used extensively to represent calmness and responsibility.



I

Indigo is a dark purplish-blue in colour with the hex code #330099. It represents an inner awareness of spirituality, and allows for deep focus during meditation.



V

Violet is the color of Roman emperors, monarchs and princes with the hex code #803790, purple and violet are often associated with royalty, nobility, luxury, power, and ambition.
